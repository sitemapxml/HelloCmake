# HelloCmake
 C HelloWorld program with cmake

Build tools:

 - [gcc](https://gcc.gnu.org/)
 - [cmake](https://cmake.org/)

<br>
Editor:

 - [Geany](https://www.geany.org/)

<br>
Run cmake:

```
cmake -S source -B build
```

Compile with gcc:

```
gcc -Wall ./source/hello.c -o ./build/hello
```

Run hello binary:

```
./build/hello
```
Output:

```
Hello, World!
```
